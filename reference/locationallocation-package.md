# locationallocation: Maximal Coverage Location-Allocation Using Geospatial Data

Functions to perform Maximal Coverage Location-Allocation analyses using
geospatial data. It is designed to help decision-makers optimally
allocate resources and facilities in various contexts, such as public
health, emergency response, and urban planning.
