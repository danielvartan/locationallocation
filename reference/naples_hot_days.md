# Hot days in the city of Naples, Italy

A [`RasterLayer`](https://rdrr.io/pkg/raster/man/raster.html) object
representing the number of hot days in the city of Naples, Italy.

## Usage

``` r
naples_hot_days
```

## Format

A [`RasterLayer`](https://rdrr.io/pkg/raster/man/raster.html) object
with 1 layer.
