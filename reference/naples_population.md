# Population density in the city of Naples, Italy

A [`RasterLayer`](https://rdrr.io/pkg/raster/man/raster.html) object
representing the population density in the city of Naples, Italy.

## Usage

``` r
naples_population
```

## Format

A [`RasterLayer`](https://rdrr.io/pkg/raster/man/raster.html) object
with 1 layer.
